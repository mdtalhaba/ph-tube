VAR
  • Var variable is functional scope and var variables that can be reassigned another value.
  • They are visible throughout the entire function, even if declared inside a block statement.
    • LET
  • Let variable is block scope and let create variables that can be reassigned another value.
  • They are visible only within the block in which they are declared, enclosed by curly braces.
    • CONST
  • Const variable is block scope and const variables that cannot be reassigned another value.
  • They are visible only within the block in which they are declared, enclosed by curly braces.

    • NULL
  • Null is a reserve keyword of javascript and it is special value that represents an empty or unknown value, Use null when you want to represent the intentional absence of a value
    • UNDEFINED
  • Undefined is a reserve keyword of javascript and it is a built-in value in JavaScript that helps indicate when a variable or property has not been assigned a value or when a value is not available

    • REST API
  • REST API is a API interface that two computer systems use to exchange information securely over the internet and it is a way of accessing web services in a simple and flexible way without having any processing.
